Searching

You can use Python's regex syntax to search for anything in the logs. Note that this interface does not support plus signs (+) and ampersands (&). If you are unsure of your syntax, check it with this online tool.

Adding Logs

FLS looks in the default log directory (/var/log/remotesystems/) and adds every file found to it's "Host" selector. Note that the user which Apache runs under must have read access to these files for FLS to search them!

Allow Apache To Read Logs

sudo chmod o+r -R /var/log/remotesystems/

Options

  • Update: Updates the results based on the current query every 10 seconds.
  • Top to Bottom: By default, FLS starts at the bottom of the log file(s) when searching. With this option enabled, it searches from the top to bottom.
